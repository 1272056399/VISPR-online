# VISPR-online
VISPR-online is a web-based interactive framework for CRISPR screens visualization.

Deployment
-------
##### Step1:
Install Python 3.5. Create development environment with
```
$ pip install -r requirements.txt
```
##### Setp2:
Start the application with
```
$ ./run.py
```
